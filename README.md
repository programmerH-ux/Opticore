# OptiEngine Editions & Pricing
OptiEngine is a high-performance optimization engine written in Rust with Python bindings, designed for speed, reliability, and real-world deployment.
It is available in Free and Pro editions.

License: Apache 2.0
Best for: Students, hobbyists, open-source projects
Includes:
Gradient Descent optimizer
Core optimization utilities
Full source code access
Free for personal and commercial use
Price: Free
